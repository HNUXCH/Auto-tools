# 图片工作流（Windows 图形版）

## 工作流步骤

可勾选以下步骤并按顺序执行：

1. 批量重命名
2. 批量文字水印
3. 批量压缩输出

## 特点

- 🛡️ 原图不会被修改（在临时目录操作）
- 🚫 支持中途取消
- 🔤 自动检测系统中文常用字体（微软雅黑、黑体、宋体等）
- ✅ 单文件容错：个别图片出错不影响整体
- 📋 参数统一在一个表单中填写（减少弹窗）
- 📊 实时进度条，分阶段显示

## 运行

```bash
pip install -r requirements.txt
python main.py
```

然后：选择输入目录 → 选择输出目录 → 填写参数表单 → 开始执行
