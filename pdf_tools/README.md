# PDF 处理工具（Windows 图形版）

已改为 Windows 下可在 PyCharm/其他编译器直接运行的图形界面版本。

功能：
- PDF 合并（多选）
- PDF 按页拆分
- PDF 文本提取
- PDF 页面旋转（90/180/270）

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行方式（PyCharm / 其他 IDE）

1. 打开 `pdf_tools/main.py`
2. 直接运行脚本
3. 在窗口中点击对应按钮，按弹窗选择输入/输出

## 使用流程

- **合并 PDF**：多选输入文件 -> 选择输出文件
- **按页拆分**：选择输入文件 -> 选择输出目录
- **提取文本**：选择输入文件 -> 选择输出 txt
- **旋转页面**：选择输入文件 -> 输入角度 -> 选择输出文件
