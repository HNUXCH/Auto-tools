# Windows 图片批量重命名工具

一个可在 Windows 上直接运行的 Python 小工具。

运行后流程：
1. 弹窗选择图片文件夹
2. 弹窗输入前缀（可空）
3. 弹窗输入后缀（可空）
4. 弹窗输入起始编号（如 `00`、`000`、`0000`）
5. 自动按 `前缀_编号_后缀.扩展名` 重命名

## 命名规则

- 分隔符固定为 `_`
- 前缀可空
- 后缀可空
- 编号位数由起始编号决定（例如输入 `000`，则会生成 `001`、`002`...）
- 保留原图片扩展名（统一为小写）

示例（前缀=`image`，后缀=`cat`，起始编号=`000`）：
- `image_000_cat.jpg`
- `image_001_cat.jpg`
- `image_002_cat.png`

如果前缀和后缀都为空，名称会变成：
- `000.jpg`
- `001.jpg`

## 运行方式（PyCharm / 其他 IDE）

- 使用 Python 3.10+ 打开项目
- 直接运行 `main.py`

## 支持格式

`.jpg` `.jpeg` `.png` `.bmp` `.gif` `.webp` `.tif` `.tiff`
